# Veri Yapıları Ödevi: Arama ve Sıralama

Bu proje, temel algoritmaların Python uygulamalarını ve performans analizlerini içerir.

## 📊 Algoritma Analizi (Big-O)
| Algoritma | Ortalama Karmaşıklık | Mantık |
| :--- | :--- | :--- |
| **Binary Search** | $O(\log n)$ | Sıralı listeyi ikiye bölerek arar. |
| **Linear Search** | $O(n)$ | Tek tek kontrol eder. |
| **Merge Sort** | $O(n \log n)$ | Böl ve yönet prensibiyle çalışır. |
| **Bubble Sort** | $O(n^2)$ | Komşu elemanları kıyaslar. |

## 💡 Karşılaştırma
Büyük veri setlerinde **Merge Sort** en hızlı sıralama yöntemidir. Arama yaparken liste sıralıysa **Binary Search** kullanmak büyük zaman kazandırır.
